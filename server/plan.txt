SPA приложение с клиент-серверной архитектурой.
1. Разработал Авторизацию/Регистрацию, JWT.
2. Личный кабинет пользователя(забронированные коттеджи, лайки,
избранное) с возможностью редактирования.
3. Панель администратора (Таблица коттеджей с выпадающим списком
бронирований для каждого коттеджа, отмена бронирования,
редактирование коттеджа)
4. Страница с доступными коттеджами ( Поиск, Сортировка, Фильтрация,
Пагинация).
5. Страница коттеджа ( Возможность забронировать коттедж, оставить отзыв,
система лайков, рейтинг коттеджа).
